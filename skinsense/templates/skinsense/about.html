About SkinSense

SkinSense is an intelligent, AI-powered skin condition diagnosis system designed to assist individuals in understanding their skin health through image analysis. By leveraging deep learning models, the system can detect common skin conditions including acne, eczema, and vitiligo.

The platform allows users to upload a facial image securely, where the model evaluates the condition and provides instant results such as predicted skin issue, confidence level, and risk assessment. Additionally, the system integrates the Gemini API to generate personalized skincare advice for each user.

Our Mission

To create a reliable, accessible, and AI-driven digital health solution that promotes early awareness and preventive skincare practices through intelligent analysis and personalized insights.

Technologies Used

  • Frontend: HTML, CSS, JavaScript
  • Backend: Django REST Framework
  • AI Model: ResNet50 (TensorFlow)
  • Database: MySQL
  • API Integration: Gemini AI API for advice generation
